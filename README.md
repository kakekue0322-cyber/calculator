## 特徴 / Features

| 機能 | 説明 |
|---|---|
| 計算履歴 | 計算するたびに式と結果が残る（最大100件） |
| リアルタイム表示 | 入力中の計算式を大きく表示 |
| 履歴の再利用 | 過去の計算結果をタップして再利用 |
| インストール不要 | URLを開くだけですぐ使える |
| PWA対応 | ホーム画面に追加してアプリとして使える |
| オフライン動作 | 一度読み込めばネットなしでも動作 |
| スマホ最適化 | iPhoneでも快適に動作するUI |

## 使い方 / How to Use

1. [ライブデモ](https://kakekue0322-cyber.github.io/calculator/) を開く
2. 普通の電卓と同じように計算する
3. `=` を押すと計算式と結果が上の履歴エリアに残る
4. 過去の履歴をタップすると、その結果を次の計算に使える

**スマホのホーム画面に追加する方法**
- iPhone：Safari で開く → 共有ボタン →「ホーム画面に追加」
- Android：Chrome で開く → メニュー →「ホーム画面に追加」
